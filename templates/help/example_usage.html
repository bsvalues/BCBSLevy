Contextual Help Example

This example shows how to implement contextual help bubbles with AI-powered explanations in the TerraFusion platform.

Basic Usage

Simply add a data-help-topic attribute to any element that needs an explanation. The value should match one of the predefined help topics.

Example 1: Tooltip Help

When calculating levy rates, the levy rate is multiplied by the assessed value to determine the levy amount.
