Import Guidelines
Property Data Format
  • CSV file with headers
  • Required columns: property_id, assessed_value, tax_code
  • Optional columns: address, owner_name, property_type
  • Maximum file size: 10 MB
Tax District Data Format
  • Supported formats: CSV, XML, TXT, Excel
  • Required columns: tax_district_id, district_name, levy_code
  • Optional: year (defaults to current year)
  • Maximum file size: 5 MB
